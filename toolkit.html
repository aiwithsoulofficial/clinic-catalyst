Getting Started

Two ways to install - pick the one that fits you

Claude Desktop Users

Step 1

Sign up at claude.ai if you haven't already. These skills work with Claude Pro, Claude Team, or Claude Enterprise.

Step 2

Enable MCP connectors in your Claude settings - Canva (for Creative Assets), Gmail (for Sales & Marketing), Apify (for competitor research), and Google Calendar (for scheduling context).

Step 3

Add the downloaded skills in the Customise section of your Claude Desktop settings - exactly like we showed you on stage. Need a refresher? Grab the slides here.

Step 4

Start prompting. Claude will use the skills as instructions. Try:

"Audit this Instagram caption for slop patterns"

"Create a testimonial post for my clinic using Canva"

"Draft a follow-up sequence for my latest enquiries"

Claude Code Users

For developers and technical users - install as slash commands:

mkdir -p ~/.claude/skills/stoptheslop/references ~/.claude/skills/sales-marketing-clinics ~/.claude/skills/creative-assets-clinics

Save each file as SKILL.md in its skill folder. Reference files go in references/.

Then use /stoptheslop, /creative-assets-clinics, or /sales-marketing-clinics.
